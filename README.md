# Install

```
1. git clone https://github.com/hi-one-gg/heic-to-png-using-multiProcessing.git
2. pip install pillow, pillow_heif, functools
```
</br>

# Usage
```
python heic_to_png_parallel.py {heic directory} {num_cpu_core}
```
